# Spandan

## General Purpose
Personal workspace for specialized model experiments including DOFA-CLIP, EarthDial project, and Google Earth Engine scripts for satellite data acquisition.

## Files

### `DOFA-CLIP/`
Implementation or experiments with DOFA-CLIP (Domain Focal Adaptation for CLIP) for improved satellite image understanding.

### `EarthDial/`
EarthDial project codebase for dialogue-based Earth observation analysis.

### `google_earth_engine/`
Scripts and notebooks for accessing and processing satellite imagery via Google Earth Engine API.

### `models_to_test/`
Collection of model checkpoints or configurations queued for evaluation.
