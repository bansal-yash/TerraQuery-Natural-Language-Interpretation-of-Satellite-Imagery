# SkySkript Mini

## General Purpose
Mini-project for satellite image processing with Google Earth Engine integration and super-resolution experiments.

## Files

### `sr.py`
Google Earth Engine script for downloading and processing Sentinel-1 SAR imagery with statistical analysis, dB conversion, and focal median filtering.

### `mv.sh`
Shell script for moving/organizing files within the project structure.

### `images/`
Directory containing processed satellite imagery outputs.

### `meta/`
Metadata files associated with downloaded imagery.
