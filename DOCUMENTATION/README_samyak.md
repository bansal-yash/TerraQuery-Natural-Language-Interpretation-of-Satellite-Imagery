# Samyak

## General Purpose
Personal workspace containing multiple sub-projects related to satellite imagery captioning, VQA, and fine-tuning experiments with EarthMind, GeoChat, and VRSBench datasets.

## Files

### `EarthMind/`
Code and experiments related to the EarthMind-Bench satellite imagery dataset.

### `GeoChat/`
Experiments with GeoChat vision-language model for geospatial understanding.

### `GeoPixel/`
GeoPixel-related experiments for pixel-level geospatial analysis.

### `qwen_finetune/`
Qwen model fine-tuning scripts and configurations specific to samyak's experiments.

### `VRSBench/`
Work with VRSBench (Visual Reasoning for Satellite imagery) dataset.

### `Grounded-Segment-Anything/`
Local clone of Grounded-SAM repository for grounding and segmentation experiments.

### `gd_fine_tune/`
GroundingDINO fine-tuning experiments.

### `ok.py`
Quick testing or utility script.

### `P0003_0002.png`
Standard test image tile.

### `VRSBench_train.json`
Training data in VRSBench format.

### `annotated_buses.jpg`
Example annotation visualization.

### `bin/`
Binary files or compiled utilities.
