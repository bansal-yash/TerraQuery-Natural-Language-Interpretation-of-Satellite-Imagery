# Report Images

## General Purpose
Collection of visualization outputs and comparison images used for documentation, reports, and presentations showing model performance on various tasks.

## Files

### `normal_rgb_bbox.png`
Bounding box detection results on normal RGB satellite imagery.

### `out_base_ships.png`
Base model performance on ship detection task.

### `out_base_false_ships.png`
Base model false positives/negatives in ship detection.

### `out_ships.png`
Fine-tuned model ship detection results.

### `out_false_ships.png`
Fine-tuned model false detections analysis.

### `totest/`
Directory of additional test cases and experimental visualizations.
